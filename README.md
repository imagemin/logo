# logo

> The imagemin logo

Easily include the imagemin logo on your site without editing any images. This 
component is built on the [logo/logo](https://github.com/logo/logo) base.


## Usage

```html
<div class="logo" data-logo="imagemin">
	<span>imagemin</span>
</div>
```
